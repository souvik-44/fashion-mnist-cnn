# Fashion-MNIST CNN

A CNN model to classify Fashion-MNIST images (10 classes) using TensorFlow/Keras.

## Workflow
- Data preprocessing (normalize, reshape)
- CNN model training
- Evaluation (accuracy, confusion matrix, classification report)

## Tech Stack
- Python
- TensorFlow/Keras
- NumPy, Matplotlib
- scikit-learn

## How to run
Open `fashion_mnist_cnn.ipynb` in Jupyter and run all cells.
